USERGUIDE:
IN VSC: HOMETOOLBAR -> FIND TERMINAL -> NEWTERMINAL OR ( PRESS CTRL+ SHIFT +` ) for new terminal
1.Environment Setup
The system requires Python 3.8 or higher.
Open your Terminal (Command Prompt or PowerShell) and navigate to the project root directory.
Install the required dependencies by running:

pip install -r requirements.txt

(Dependencies include: streamlit, numpy, pandas, sympy, matplotlib)
2.Running the Application
To launch the interface, run the following command in the terminal:

streamlit run app.py

The application will automatically open in your default web browser at http://localhost:8501.
